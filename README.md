# AppGallery

Crie um app em ANDROID NATIVO com o nome "App Gallery - X" onde X, representa seu primeiro Nome.

Para acessar a API use uma dessas 2 autorizações:

Authorization
"Client-ID bxxxxxxxxxxxxxxxc"
ou
"Client-ID dxxxxxxxxxxxxxxxe"

Aplicar as seguintes regras: 

                1 O APP deverá:

                               ○ Consumir a API https://api.imgur.com/3/gallery/search/?q=YYYY, onde YYYY é o parâmetro a ser consultado, por exemplo "cats"

                               ○ Capture os seguintes valores da resposta:

                                               § Nome de Exibição

                                               § Seguidores;

                                               § Descrição;

                                               § URL da Imagem.

                2 Crie uma tela com os seguintes componentes :

                               a. Uma label com a Descrição: "Filtrar por:";

                               b. Um input para o usuário informar o dado a ser consultado;

                               c. Um botão com a label "Pesquisar";

                               d. Um componente de lista;

                               e. Componentes de Imagem para exibir cada foto.

                3 Comportamento do App:

                               a. O usuário irá inserir no input o texto de pesquisa, por exemplo "Cat"

                               b. O usuário irá clicar no botão "Pesquisar"

                               c. O App irá consultar a api e retornar os resultados.

                               d. O App irá filtrar os resultados pelo tipo de imagem "JPG" ou "JPEG"

                               e. O App irá listar as fotos;

                                               i. As fotos deverão ser exibidas de forma a não travar ou sobrecarregar o app;

                                               ii. Pode-se usar bibliotecas de terceiros para descarregar as imagens.

                               f. Ao clicar em uma foto, o app deverá exibir as seguintes informações:

                                               i. Nome de Exibição

                                               ii. Seguidores

                                               iii. Descrição
             
                4 Personalize a exibição dos componentes para modelos atuais de UX.

                5 Personalize a apresentação da interface para ser amigável e intuitiva ao usuário final.

               6 A Interface deverá ser responsiva para diversos tamanhos de dispositivos celulares.
